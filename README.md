# js_flappy_bird</br>
## 项目描述</br>
利用js实现简单的flappy bird游戏</br>
素材使用原游戏图，页面基于css实现</br>
使用main game.html可试玩，鼠标点击进行操作</br>
程序模块实现说明：[JS+CSS实现小游戏Flappy bird](https://blog.csdn.net/qq_41733192/article/details/105790516)</br>
